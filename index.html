Pulsar orbit plotter

This is a little experiment I've done to test JavaScript animations with the D3 library. It shows the orbital movement of a pulsar and (qualitatively) the change of the observed rotation period and the delay in the arrival times of the pulses due to finite light travel time.

Insert eccentricity and orientation, then click Refresh page. Then... Start!

Don't make it last too long. Memory efficiency is not its thing.

Also, time and orbital dimensions are in arbitrary units for the time being.

I will update the code to accommodate for these "uninportant" details later :).
